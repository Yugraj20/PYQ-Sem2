Web Based Programming (WBP) - Important Questions

1. Arrays & Their Functions

  • Concepts: What do you understand by an Associative array? How is an associative array different from an indexed array?
  • Coding: Write a short note on associative arrays. Create an associative array using countries as keys and cities as values, transform it into a 2D array, and display the data in a tabular form.
  • Functions: Explain explode(), krsort(), and count() functions with examples.

2. Functions (Call by Value vs. Call by Reference)

  • Concepts: Differentiate between Call by Value and Call by Reference concepts with suitable examples.
  • Coding: Write a user-defined function for swapping two variables using call by value and call by reference. Differentiate the output produced by both methods.

3. PHP Errors & Types

  • Concepts: What are the different types of errors in PHP? Explain by citing appropriate examples for each type.
  • Related: Differentiate between exceptions and errors.

4. Forms, Validations & Methods (GET/POST)

  • Concepts: What are the differences between GET, POST, and REQUEST methods?
  • Coding (Form): Write a PHP code to design a "Contact Us" form and provide validation on the email and name fields.
  • Coding (Practical Application): Create a registration form (fields: Name, Gender, Address, Hobbies, City, DOB). Write a PHP script to submit this data and print the received data on another page.

5. State Management (Sessions & Cookies)

  • Concepts: What are the differences between Cookies & Sessions? Differentiate between state management techniques in PHP.
  • Coding (Sessions): Explain Sessions and write a PHP script to count the number of visits to a web page by a user using a session variable (in 24 hours).
  • Coding (Advanced): Create a registration form for a bank portal where the session should expire if the system is idle for 1 minute.

6. File & Directory Handling

  • Concepts: What is File Handling in PHP? What are the different file opening modes used with the fopen() function?
  • Functions: Explain the following file handling functions: fread(), fgets(), and unlink().
  • Coding (Read/Write): Write a PHP script to open, close, read, and write into a file line by line, displaying it on a webpage.
  • Coding (Upload): Write a program using PHP to upload and download an image. Implement validations (e.g., must be format *jpg, *png, *jpeg, and must not be greater than 1MB).

7. Server Architectures

  • Concepts: Differentiate between WAMP, LAMP, MAMP, and XAMPP.
  • Related Server Concepts: What are the advantages of PHP? Differentiate between Client-Side Scripting vs. Server-Side Scripting, and Static Website vs. Dynamic Website.

8. Include, Require & Super Globals

  • Concepts: Differentiate between include(), include_once(), require(), and require_once().
  • Super Globals: What are Super Global Variables in PHP? Explain them in detail (especially $_REQUEST, $_GET, $_POST).
